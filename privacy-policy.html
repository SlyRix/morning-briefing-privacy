Privacy Policy for Morning Briefing Skill

Effective Date: July 22, 2025

Overview

This privacy policy describes how the Morning Briefing Alexa Skill ("the Skill") handles user information.

Information Collection and Use

The Morning Briefing Skill does NOT collect, store, or transmit any personal user information. The Skill:

  • Does not access or store any personal data
  • Does not use account linking
  • Does not collect voice recordings beyond standard Alexa processing
  • Does not access personal calendars, contacts, or location data directly through the skill
  • Only processes voice commands to provide morning briefing information

Data Sources

The Skill retrieves public information including:

  • Weather data for specified locations
  • Public stock market information
  • Pre-configured calendar events (if user has set up integration)

Amazon Alexa Data Handling

This Skill uses Amazon's standard Alexa voice processing. Please refer to Amazon's Privacy Policy for how Amazon handles voice interactions: https://www.amazon.com/privacy

Third-Party Services

The Skill may retrieve information from:

  • Weather services (OpenWeatherMap)
  • Financial data providers
  • Calendar services (if configured)

These services have their own privacy policies which govern their data handling.

Data Storage

This Skill does not store user data on external servers. Any temporary processing is done solely to provide the requested morning briefing service.

Children's Privacy

This Skill is not directed at children under 13 and does not knowingly collect information from children under 13.
